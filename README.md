# Ying's Learning Notes

Personal blog for technical notes. Built with [Jekyll](https://jekyllrb.com/) and the [So Simple](https://github.com/mmistakes/so-simple-theme) theme, hosted on [GitHub Pages](https://pages.github.com/).

## Local Development

```bash
bundle install
bash run.sh        # preview at http://localhost:4001 (includes drafts)
```
